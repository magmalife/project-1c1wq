Hevyless API Free

Sync your workouts without a Hevy Pro subscription by using the Hevyless Mock API.

How it works

The Hevyless API allows you to bypass the Hevy Pro API requirement by perfectly mocking the official API's responses. It scrapes your public Hevy web profile to fetch your latest workout and transforms it into the exact format that hevy2garmin expects.

You just need to ensure your Hevy profile is set to Public and enter your username below.

Make sure your profile visibility is set to Public in the Hevy app settings.

Vercel Setup Instructions

Since you are viewing this dashboard, the Hevyless API is already running natively as part of your application! To tell hevy2garmin to use it, you just need to override the API URL.

  1. Go to your Vercel Project Settings → Environment Variables.
  2. Add HEVY_API_KEY_URL and set the value to:
    https://<your-project>.vercel.app/api/hevyless/v1
  3. Add HEVY_API_KEY and set it to any dummy value (e.g. hevyless).
  4. Trigger a sync!
